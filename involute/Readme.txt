１）ソフト紹介
　平歯車、内歯車、歯直角はすば歯車(インターナル)、軸直角はすば歯車
　(インターナル)の諸元を入力し、インボリュート曲線（円弧補間）を
　Jw_cad for Windows(以下Jww)の座標ファイル形式またはDXF形式で出力
　するソフトウェアです。

１）インストール
　Inv093.LZHを適当なフォルダに解凍して下さい。必要なファイルは以下です。
　解凍後、Setup.exe をダブルクリックすると対話形式でインストールできます。

　　1)dxf     .dat ... DXF作成用データ
　　2)Involute.exe ... 本体
　　3)Inv     .cnt ... ヘルプ補助
　　4)Inv     .HLP ... ヘルプ本体
　　5)Readme  .txt ... 導入案内ファイル
　　6)Setup   .exe ... インストール
　　7)Setup   .inf ... インストール情報

　　※以下は外変版に必要なファイルでインストーラではインストールされません。
　　8)Inv.bat      ... 外変用BATファイル
　　9)InvJww.txt   ... 外変版導入案内ファイル
　 10)InvJww.exe   ... 外変版本体

　※外変版のインストールはエクスプローラなどを使い 8) 9) 10)のファイルをJww
　　のインストールフォルダにコピーして下さい。

２）アンインストール
　アプリケーションの追加と削除から「歯車作成 Inv」を選んで実行して下さい。

　※外変版は8) 9) 10)のファイルを削除して下さい。

３）起動・設定
　Involute.exeをダブルクリックすると起動できます。
　※操作方法は同梱のヘルプをご覧ください。(Invの中から起動できます)
　※外変版は InvJww.txt をご覧ください。
